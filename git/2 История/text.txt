урок 3. Как менять историю: git reset/revert 

    git revert (sha) позволяет отменить изменения в коде с созданием нового комита  
    если мы не хотим видеть последний комитт, тогда используем  git reset  с ключами : --hard, --soft, --mixed
        git reset HEAD~(количество коммитов на которое нужно откатиться) 
        git reset (ahs)  откатит до указанного комита 
        --hard : удалит как комит так и внесенные изминения
